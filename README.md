# jsc_sft_study
This study investigated solitary fibrous tumor (SFT), a rare mesenchymal neoplasm. Samples were collected from the tumor core (LT) and peritumoral tissue (LZ). The scATAC-seq dataset included seven samples, comprising four LT samples and three LZ samples.The scRNA-seq dataset included nine samples, comprising six LT samples and three LZ samples.
